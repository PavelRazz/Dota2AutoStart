# Хлопковый запуск Dota 2
Программа для автоматического запуска Dota 2 при обнаружении двойного хлопка через микрофон.

# Возможности
Автоматическое определение аудиоустройств

Настройка чувствительности и частотного диапазона

Обнаружение одиночных и двойных хлопков

Запуск Dota 2 через Steam при обнаружении двойного хлопка

Безопасное завершение по Ctrl+C

# Требования
Python 3.7+

Установленные библиотеки (см. Установка)

Микрофон

Установленный Steam и Dota 2

Установка
Клонируйте или скачайте репозиторий

Установите зависимости:

## bash

pip install -r requirements.txt
 
Файл requirements.txt должен содержать:

text
clapDetector  # или другая библиотека для детектирования хлопков
Убедитесь, что у вас установлен Steam и Dota 2

# Настройка
Запустите программу один раз, чтобы увидеть список доступных аудиоустройств:

## bash

python Autostart.py

Отредактируйте файл Autostart.py и укажите индекс вашего микрофона:

## python

MIC_INDEX = 1  # замените на номер вашего микрофона из списка
При необходимости настройте параметры чувствительности:

python
threshold_bias = 2500  # чувствительность (меньше = более чувствительно)
lowcut = 100           # нижняя граница частот (Гц)
highcut = 4000         # верхняя граница частот (Гц)

# Использование
Запустите программу:

## bash
## python Autostart.py
Программа выведет список доступных аудиоустройств и начнёт ожидать двойной хлопок.

Как это работает:
Программа непрерывно слушает микрофон

При обнаружении одного хлопка выводится сообщение "Один хлопок обнаружен"

При обнаружении двух хлопков в быстрой последовательности:

Выводится сообщение "Два хлопка обнаружены! Запускаю Dota 2..."

Запускается Steam с параметром для запуска Dota 2 (app ID: 570)

Программа завершает работу

Пути запуска
По умолчанию программа использует стандартный путь установки Steam:

text
C:\Program Files (x86)\Steam\steam.exe
Если у вас Steam установлен в другом месте, отредактируйте строку запуска в коде.

Завершение работы
Для ручного завершения программы нажмите Ctrl+C.

# Возможные проблемы
"Audio init failed" - проверьте индекс микрофона и подключение устройства

Слишком много ложных срабатываний - увеличьте значение threshold_bias

Хлопки не обнаруживаются - уменьшите threshold_bias или проверьте микрофон

Не запускается Dota 2 - проверьте путь к Steam и наличие установленной игры

Примечание: Для работы программы необходим подключенный и настроенный микрофон. Рекомендуется тестировать в тихой обстановке для предотвращения ложных срабатываний.
